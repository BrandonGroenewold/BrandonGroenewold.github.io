Projects

Here are analytical projects that I have completed

  • Minneapolis Crime Dashboard

    Here is an interactive map that shows the neighborhoods of Minneapolis and a crime breakdown for the area you have selected. Skills/Libraries Used: Python, JavaScript, Flask, Leaflet, Jupyter Notebook, HTML, and CSS.

  • ETL (Extract Transform Load)

    This is a project showing data clean up, web scraping, and transforming into useful and useable data. Skills/Libraries Used: Python, Jupyter Notebook, and Web Scraping.

  • Traffic Data

    Traffic Data from 2021 was loaded into a jupyter notebook to clean up and use to analyze trends across the US. Displays graphs and visiuals to showcase any findings we have found. Skills/Languages Used: Python, Jupyter Notebook, gmaps, Matplotlib, and Seaborn.

  • ETL (Extract Transform Load)

    We performed the ETL process using two datasets gathered from Kaggle and Wikipedia. To perform this process, we were required to make use of Pandas, SQLAlchemy, BeautifulSoup, and a few more Python libraries. The two datasets were a CSV downloaded from Kaggle and information scraped from a Wikipedia article. Skills/Languages Used: Python, Jupyter Notebook, gmaps, Matplotlib, and Seaborn.

  • Employee Database

    Using PostgreSQL employees, managers, departments, titles, and salary where loaded in to establish relations between the various tables making it ready for querying and to make employee information readily availible Skills Used: PostgreSQL, Qickdatabasediagrams.com, ERD

  • Machine Learning project (coming soon)

    To be showcased

  • API Vacation

    APIs were used to analyze temperature, humidity, cloudiness, and wind speed across northern and southern hemispheres, and then used a heatmap and specific weather conditions to find an ideal vacation spot. Skills/Languages Used: Jupyter Notebook, Python, APIs, Heatmap, Numpy, Pandas, and Matplotlib.

  • Big Data

    Google Colab was used to analyze two datasets from Amazon with over 1.5 million rows. An ETL was performed to make the two datasets into a DataFrame and analyze whether reviews from Amazon's Vine program are trustworthy. Skills/Languages Used: PySpark, Google Colab, Python, SQL, PostgreSQL
